# 测试规范 — 自动化数学建模软件

## 测试数据集

### test_data/regression_sample.csv
房价预测数据，包含：
- 面积（数值）、卧室数（数值）、房龄（数值）、地段（文本）、房价（数值，目标列）
- 刻意加入 5% 缺失值、2 个异常值（面积=999999）
- 约 200 行

### test_data/classification_sample.csv
鸢尾花分类数据（Iris 变体）：
- 花萼长、花萼宽、花瓣长、花瓣宽（数值）、品种（文本，目标列）
- 刻意加入 3 个品种拼写错误、5 个缺失值
- 约 150 行

### test_data/clustering_sample.csv
客户分群数据：
- 年收入、消费分数、年龄、会员年限（数值）
- 无目标列（无监督学习）
- 约 300 行
