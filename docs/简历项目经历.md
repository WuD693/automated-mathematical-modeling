# 简历项目经历 — 自动化数学建模平台

## 项目名称

**自动化数学建模平台（AutoMathModel）** — 零代码数据科学分析工具

## 项目简介

独立设计并开发了一款 Windows 桌面端自动化数学建模软件，面向无编程基础的数据分析需求者。用户仅需上传数据集，系统自动完成数据清洗、模型选择与训练、Nature 期刊风格可视化及中文结论输出，全程无需编写任何代码。

## 技术栈

Python · Streamlit · Pandas · NumPy · Scikit-learn · Matplotlib · Seaborn · PyInstaller · Git · GitHub

## 核心工作

- **架构设计**：采用模块化分层架构，将数据加载、清洗、建模、可视化、结论生成拆分为 5 个独立模块，模块间通过统一接口解耦，支持灵活替换底层实现（如 MATLAB MCP 与 matplotlib 双引擎切换）
- **自动建模引擎**：实现问题类型自动判定（回归/分类/聚类），集成 11 种机器学习模型的自动训练与交叉验证对比，自动选出最优模型；通过特征重要性分析提取关键影响因素
- **出版级可视化**：按照 Nature 期刊规范定制 matplotlib 全局样式（去除冗余边框、科学色板、Arial 字体），实现 7 种图表类型并支持 300 DPI 高清导出；解决 Windows 中文字体渲染问题
- **数据清洗管道**：实现缺失值自动填充（数值→中位数、分类→众数）、IQR 异常值检测与 Winsorize 截断、文本列智能编码（低基数 One-Hot / 高基数 Label Encoding）、重复行检测
- **中文结论生成**：设计规则引擎将模型指标（R²、准确率、轮廓系数）转化为通俗中文描述，自动提取回归方向性（正/负相关）与建模建议（线性 vs 非线性）
- **用户体验**：设计 6 步向导式流程 UI（淡蓝色主题），实现一键操作、加载动画、中文清洗报告、图表下载等功能，降低非技术用户使用门槛

## 项目成果

- 完整打通 上传→清洗→建模→可视化→结论 全流程，支持 CSV/Excel 格式数据
- 覆盖 3 种分析类型（回归/分类/聚类），11 种机器学习模型
- 生成符合 Nature 期刊风格的高清图表，支持 PNG 导出
- 通过批处理脚本实现一键安装 Python 环境与依赖，零门槛启动
- 项目已开源并推送至 GitHub，配备完整的需求/技术/设计规范文档与测试数据

## 面试可能追问的准备

| 问题 | 回答要点 |
|------|----------|
| 为什么选 Streamlit？ | 专为数据科学场景设计，自带表格/图表/上传组件，开发效率比 PyQt/Flask 高 3-5 倍 |
| 怎么判断问题类型？ | 根据目标列的数据特征：连续数值且多样值→回归；离散类别或少样值→分类；无目标列→聚类 |
| 怎么保证模型可靠性？ | 3 折交叉验证 + 多模型对比（回归 5 个、分类 4 个、聚类 2 个），选最优而非固定模型 |
| 最大的技术难点？ | 清洗器 One-Hot 编码导致目标列丢失→通过 y_original 参数传递原始列解决；中文在 matplotlib 中显示方块→注册微软雅黑字体解决 |
| 怎么让别人使用？ | setup.bat 自动检测/安装 Python + 依赖，实现零门槛启动；项目已推送 GitHub 开源 |
