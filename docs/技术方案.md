# 技术方案 — 自动化数学建模软件

## 技术选型

### 总体方案：Python + Streamlit → PyInstaller 打包

| 层级 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| UI 框架 | Streamlit | ≥1.28 | 专为数据科学设计，自带表格/图表/上传组件，开发效率极高 |
| 数据处理 | pandas | ≥2.0 | Python 数据分析事实标准 |
| 数值计算 | numpy | ≥1.24 | 底层数组计算 |
| 机器学习 | scikit-learn | ≥1.3 | 最成熟的 ML 库，覆盖回归/分类/聚类 |
| 可视化 | MATLAB MCP Core Server | — | MATLAB 出版级图表，Nature 期刊标准，通过 MCP JSON-RPC 协议调用 |
| Excel 读取 | openpyxl | ≥3.1 | .xlsx 文件解析 |
| 打包分发 | PyInstaller | ≥6.0 | Python 项目打包为单文件 .exe |

### 已拒绝的方案

- **PyQt / Tkinter**：UI 开发效率低，样式定制复杂
- **纯 Web 应用（Flask + 前端）**：需用户配置环境，本地部署门槛高
- **Electron**：体积大，与 Python 数据科学生态集成困难

## MATLAB MCP 集成方案

### 通信架构

```
Python (Streamlit)
    │
    ├─ matlab_client.py ── stdio JSON-RPC ──► matlab-mcp-core-server-win64.exe
    │                                               │
    │                                               ▼
    │                                          MATLAB Engine
    │                                               │
    │                                               ▼
    │                                    出版级图表 (PNG/SVG)
    │                                               │
    ◄──────────── base64 图片数据 ──────────────────┘
```

### 工作流程
1. 用户触发可视化 → visualizer.py 构造 MATLAB 绘图代码
2. matlab_client.py 通过 stdio 发送 `tools/call` 请求到 MCP 服务器
3. MATLAB 引擎执行绘图，返回 base64 编码的 PNG 图片
4. visualizer.py 解码图片字节 → Streamlit 渲染展示
5. 支持 PNG/SVG/PDF 导出

### 优缺点
- **优**：MATLAB 图表质量业界顶级，Nature/Science 期刊首选，中文渲染成熟
- **缺**：需安装 MATLAB Runtime 或完整 MATLAB；进程间通信有延迟

### 降级方案
当 MATLAB MCP 不可用时，自动退回到 matplotlib + seaborn 方案。

## 打包方案

使用 PyInstaller 打包为单个 .exe：

```bash
pyinstaller --onefile --windowed --name "自动数学建模" app.py
```

- `--onefile`：合并为单个 exe
- `--windowed`：不显示命令行窗口
- `--add-data`：打包静态资源（CSS、测试数据）
- Streamlit 应用通过 `streamlit run` 子进程启动，或使用 `streamlit-frameless` 方案

由于 Streamlit 本质是 Web 应用，打包后的 exe 会自动启动本地服务器并在浏览器打开。
