# 需求文档 — 自动化数学建模软件

## 项目背景

大多数学建模工具（如 MATLAB、Python 脚本）需要编程基础，普通用户（业务人员、学生、科研工作者）使用门槛高。本项目旨在提供一款**零代码**的自动化数学建模工具，用户只需上传数据即可获得专业分析结果。

## 目标用户

- 无编程基础的数据分析需求者
- 需要快速探索数据的业务人员
- 学生和科研工作者（需要发表级别图表）
- 操作系统：Windows 10/11

## 功能需求

### FR-01：数据上传
- **优先级**：P0（必须有）
- 支持 CSV（.csv）和 Excel（.xlsx/.xls）格式
- 支持拖拽上传或点击选择文件
- 单文件大小限制 100MB
- 自动检测文件编码（UTF-8、GBK 等）

### FR-02：数据预览
- **优先级**：P0
- 上传后展示前 10 行数据表格
- 显示行数、列数、每列数据类型
- 显示基本统计量（均值、标准差、最大/最小值）

### FR-03：自动数据清洗
- **优先级**：P0
- 缺失值自动检测与填充（数值列→中位数，分类列→众数）
- 异常值检测（IQR 方法），标记后由用户决定保留或剔除
- 文本列自动编码（One-Hot 或 Label Encoding）
- 完全重复行自动删除
- 每步操作生成可读的清洗报告

### FR-04：自动数学建模
- **优先级**：P0
- 自动识别问题类型（回归/分类/聚类）
- 自动训练多个模型并对比
- 展示 Top 3 模型的性能指标
- 用通俗中文解释指标含义

### FR-05：Nature 风格可视化
- **优先级**：P0
- 数据探索图：直方图、箱线图、相关性热力图
- 模型结果图：回归拟合图、特征重要性图、聚类散点图、模型对比图
- 所有图表遵循 Nature 期刊风格
- 支持导出 PNG/SVG/PDF，分辨率 ≥ 300 DPI

### FR-06：结论自动生成
- **优先级**：P1（应该有）
- 数据整体概况描述
- Top 3-5 关键影响因素
- 模型表现直白描述
- 预测趋势（如适用）

### FR-07：UI 主题
- **优先级**：P0
- 淡蓝色主色调
- 简洁直观的步骤式导航
- 响应式布局（适配不同屏幕尺寸）
- 中文界面

### FR-08：报告导出（后续版本）
- **优先级**：P2（后续版本）
- 生成 PDF 分析报告

## 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 万行数据内清洗+建模 ≤ 30 秒 |
| 可用性 | 零编程门槛，小白可独立完成全流程 |
| 兼容性 | Windows 10/11，无需预装 Python |
| 安全性 | 所有数据处理均在本地，不上传网络 |
| 可维护性 | 模块化结构，每个模块独立可测试 |

## 不做的事（范围边界）

- 不提供手动编码/脚本编辑功能
- 不支持数据库直连
- 不做实时/流式数据分析
- 不做多人协作
- 不提供云端部署版本
- 初期不包含深度学习模型（如神经网络）
