# CLAUDE.md — 自动化数学建模软件

## 项目简介

Windows 桌面软件，零代码自动完成数据上传→清洗→数学建模→Nature风格可视化→结论输出。Python + Streamlit 架构，MATLAB MCP 生图（matplotlib 降级），PyInstaller 打包为 .exe。

## 工作约定

### 分阶段推进
- 严格按照 `docs/开发计划.md` 中的阶段顺序开发
- 每个阶段只做一件事，不跨阶段混合修改多个模块
- 当前阶段完全通过验收后，再进入下一阶段
- 开始任何工作前，先读 `docs/开发计划.md` 确认当前阶段和步骤

### 开发日志
- 每次开发会话开始时，检查 `开发日志/YYYY-MM-DD.md` 是否存在
- 不存在则按模板创建当天日志
- 会话结束时更新：今日完成事项 + 待办事项 + 遇到的问题
- 日志模板参考 `docs/开发计划.md` 中的规范

### 代码规范
- 修改代码前先读相关的 `docs/` 规范文档
- 设计变更遵循 `docs/设计规范.md` 中的配色和组件规范
- 新增函数遵循 `docs/技术方案.md` 中的模块接口约定
- 每次代码改动后运行验证，确认功能正常

### 测试数据
- 测试用数据集放在 `test_data/` 目录
- 测试标准遵循 `docs/测试规范.md`

## 常用命令

```bash
# 启动开发服务器
cd "d:/自动化数学建模"
streamlit run app.py

# 安装依赖
pip install -r requirements.txt
```

## 分享给他人

将整个项目文件夹打包为 .zip 发送。接收方解压后：

- **首次使用**：双击 `setup.bat`（自动装 Python + 依赖 + 启动）
- **已有环境**：双击 `启动平台.bat`（直接启动）
